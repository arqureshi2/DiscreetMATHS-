Lecture 1 – Logic (Simplified Summary)

Goal: Learn how to express ideas precisely using logic and understand when an argument is valid or invalid.

🔹 What is Logic?

Logic is the study of how to think correctly. It helps us understand whether a statement or an argument makes sense and is based on truth.

🔹 What is a Statement? (Also called a Proposition)

A statement is any sentence that can be clearly labeled as either true (T) or false (F)—but not both at the same time.

  • ✅ “2 + 2 = 4” → True
  • ✅ “Today is Sunday” → Depends on the day, but still a statement
  • ✅ “Grass is green” → True
  • ✅ “4 + 2 = 7” → False
  • ❌ “Close the door!” → Command, not a statement
  • ❌ “x > 2” → We don’t know what x is, so it’s not a complete statement

💡 Real-Life Example:
"Rehman lives in Canada." → This is a statement. It can be checked to be true or false.

🔹 Simple vs. Compound Statements

  • Simple statement: Just one idea (e.g., “It is raining.”)
  • Compound statement: Combines two or more ideas using logical connectives.

🔹 Negation (~)

Negation means flipping the truth value:
If p = “It is raining” (T), then ~p = “It is not raining” (F)

🔹 Symbolic Form of Statements

We use symbols instead of full sentences:

  • Let p = “It is hot”
  • q = “It is sunny”

Then:

  • “It is not hot” → ~p
  • “It is hot and sunny” → p ∧ q
  • “It is not hot but sunny” → ~p ∧ q
  • “It is neither hot nor sunny” → ~p ∧ ~q

🔹 Translating Real Sentences to Symbols

Let:

  • h = “Zia is healthy”
  • w = “Zia is wealthy”
  • s = “Zia is wise”

Then:

  • “Zia is healthy and wealthy but not wise” → (h ∧ w) ∧ ~s
  • “Zia is not wealthy but is healthy and wise” → ~w ∧ (h ∧ s)

Lecture 3: Laws of Logic

  • Identity Law: p ∧ T = p, p ∨ F = p
  • Domination Law: p ∨ T = T, p ∧ F = F
  • Double Negation: ¬(¬p) = p
  • Commutative: p ∧ q = q ∧ p

Lecture 4: Biconditional

Lecture 5: Tautology and Contradiction

Tautology: A logical statement that is always true.
Example: p ∨ ¬p

Contradiction: A logical statement that is always false.
Example: p ∧ ¬p
